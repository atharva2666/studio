# **App Name**: Atharva.AI

## Core Features:

- Personalized Intro: AI-powered intro generator: the tool will use the information to compose a personalized intro paragraph that captures Atharva's passion for coding, robotics, and AI.
- Project Showcase: Dedicated section to showcase Atharva's projects, including Olivia AI and AI testing initiatives. Include clear descriptions and links.
- Interactive Experience: Incorporate interactive elements such as animations or mini-games that reflect Atharva's skills and interests. Add hover effects to images and cards
- 3D Animated Background: Create a captivating 3D animated background that showcases abstract designs with elements related to code and AI.
- Contact Form: Allow visitors to quickly and easily reach out to Atharva for collaborations or inquiries.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey innovation, and intelligence.
- Background color: Very light gray (#F5F5F5), almost white, to ensure focus on the content and project showcases.
- Accent color: Electric violet (#8F00FF) for interactive elements, calls to action, and highlights to grab the visitor's attention.
- Headline font: 'Space Grotesk', sans-serif, to maintain the contemporary, scientific feel. Use 'Inter' for body text.
- Use sleek, modern icons with a subtle outline style.
- Implement a clean and structured layout with a clear visual hierarchy to emphasize key information and improve readability.
- Use subtle animations and transitions to create a dynamic user experience.
